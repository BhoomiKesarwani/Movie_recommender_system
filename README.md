# Movie Recommendation System

A Movie Recommendation System built using Python and PyCharm that recommends similar movies based on content similarity. The project uses movie datasets to analyze features such as genres, keywords, cast, and crew, providing personalized movie recommendations through a simple and interactive interface.

## Technologies Used
- Python
- PyCharm
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- TMDB API
- Pickle

## Features
- Search for any movie
- Get top 5 similar movie recommendations
- Display movie posters using TMDB API
- Fast and user-friendly interface
